Utilisation :
=============
 Before working:
 git pull

 After having worked:
 In the main directory, run
 ./CompileAndPush.sh "themessage" &>out.txt

 That will compile everything, update the local repository with a
 git commit -m "themessage", update the git repository and
 put the output of the commands in the file out.txt.
========================================== End ============
